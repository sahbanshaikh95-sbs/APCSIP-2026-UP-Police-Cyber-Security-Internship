# Hardware Hacking & IoT Security

## Overview

This session focused on practical demonstrations of hardware hacking, mobile exploitation concepts, IoT security, wireless attacks, and offensive security tools commonly used in penetration testing and red team operations.

---

# Mobile Hacking Concepts

The trainer demonstrated mobile attack scenarios.

## Mobile Attack Vectors

- Malicious applications
- Social engineering
- Browser exploits
- Messaging platform exploits
- Vulnerable document files
- Zero-click attacks

---

# Zero-Click Attacks

A zero-click attack requires no interaction from the victim.

The victim does not need to:

- Open a file
- Click a link
- Install an application

Examples:

- WhatsApp exploits
- iMessage exploits
- MMS exploits

### Why Dangerous?

- Difficult to detect
- No user interaction required
- Often targets high-profile individuals

# IoT Security

## What is IoT?

Internet of Things (IoT) refers to interconnected devices capable of collecting and exchanging data.

Examples:

- Smart TVs
- Smart Cameras
- Smart Locks
- Smart Meters
- Smart Appliances

---

# Common IoT Security Issues

- Weak passwords
- Default credentials
- Unencrypted communication
- Insecure firmware
- Lack of updates
- Open ports

A Raspberry Pi based Wi-Fi auditing device.

### Functions

- Wi-Fi reconnaissance
- Handshake capture
- Wireless assessment

Used for:
- Security research
- Wireless auditing
- Learning Wi-Fi security

# Bluetooth Jamming

The trainer demonstrated a self-built Bluetooth jammer.

### Purpose

Disrupt Bluetooth communication by interfering with wireless signals.

Possible impacts:

- Connection disruption
- Device unavailability
- Denial of service

---

# Flipper Zero

A portable multi-tool for hardware security testing.

### Capabilities

- RFID analysis
- NFC testing
- Sub-GHz communication
- Infrared testing
- Access control assessment

Common use cases:

- Security research
- Penetration testing
- Hardware analysis

---

# ESP32

A low-cost microcontroller widely used in IoT projects.

### Features

- Wi-Fi support
- Bluetooth support
- Low power consumption

Applications:

- IoT development
- Wireless automation
- Security testing

---

# Hashcat

A high-performance password recovery tool.

### Supported Methods

- Dictionary attacks
- Mask attacks
- Rule-based attacks
- Brute-force attacks

Popular use cases:

- Password auditing
- Security assessments
- Hash analysis

---

# CUPP (Common User Password Profiler)

Generates customized wordlists using target information.

Examples:

- Names
- Birthdates
- Nicknames
- Personal interests

Useful during password auditing engagements.

---

# Impacket SecretsDump

A tool used for credential extraction in Windows environments.

### Capabilities

- NTLM hash extraction
- SAM database extraction
- Active Directory credential dumping

Commonly used in:
- Red Teaming
- Active Directory assessments
- Post-exploitation

---

# Key Takeaways

- IoT devices significantly expand attack surfaces.
- Wireless technologies are common attack targets.
- Mobile devices are valuable targets for attackers.
- Password security remains a major weakness.
- Hardware security knowledge is becoming increasingly important for offensive security professionals.
